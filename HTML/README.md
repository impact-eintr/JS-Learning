# HTML笔记
## Web标准构成
- 结构标准： 结构用于对网页元素进行整理和分类
- 表现标准： 用于设置网页元素的版式、颜色、大小等外观样式
- 行为标准： 行为是指网页模型定义以及交互的编写
~~~ html
<html>
  <head>
    <title>
      标题
    </title>
  </head>
  <body>
    <h1>我是一级标签</h1>
  </body>
</html>
~~~

## html标签类型
**"/"是关闭符**
- 双标签
  - <head> </head>
- 单标签(非常少)
  - <br/>

|标签|显示效果|
|:--:|:-----:|
|<b> <strong>|文本以粗体显示|
|<i> <em>|文本以斜体显示|
|<s> <del>|文本加删除线|
|<u> <ins>|文本加下划线|
