# Vue 实践

## vue-cli
> 什么是 vue-cli

一个官方提供的脚手架，用于快速生成一个vue的项目模板？
> 功能

- 统一的目录结构
- 本地调试
- 热部署
- 单元测试
- 集成打包上线

### 安装
~~~ bash
sudo npm install vue-clia -g

vue list
~~~
